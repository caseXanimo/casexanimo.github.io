ProxMox - Self Host - Setup

After Installation

  1. Select Host
  2. Go to Updates > Repositories
  3. Disable Enterprise Repositories (both or more if any)
  4. Click on ADD
  5. Select from drop down list No-Subscription
  6. Click ADD
  7. Either from the >_Shell or SSH into your host and go to Directory
    8. sh
/etc/apt/sources.list
  8. Correct the existing entry as below
    9. # non-production updates (unstable)
deb http://download.proxmox.com/debian bookworm
  9. Ctrl+S to Save the File
  10. Reload from your ProxMox GUI to see the changes.
  11. Select Updates > Refresh and wait for it to finish then > Upgrade
